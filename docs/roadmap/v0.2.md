# v0.2 — Deferred candidates

> **Status**：候选清单（Tentative，等 v0.1 反馈再 prioritize）
> **Trigger**：v0.1 release 后 30-60 天，看哪些 v0.2+ 项被用户反复要求
>
> ⚠️ **本文档不是承诺**——只是把 v0.1 各处散落的 "v0.2+" flag 集中起来便于 future 评估。

---

## 1. v0.2 候选清单

按 source 分类：

### 1.1 来自 user stories (deferred)

| Story | 内容 | Trigger 条件 |
|---|---|---|
| [F2](../product/user-stories.md#f2--claude-进程僵死) | 卡死检测（JSONL mtime > X min 未更新提示） | 用户反馈"我的 session 卡死了不知道" ≥ 3 次 |
| [A1](../product/user-stories.md#a1--演示屏幕--zoom-共享时) | demo mode（屏幕共享时隐藏 last_message） | 收到 demo 泄漏 last_message 的反馈 ≥ 1 次（敏感） |
| [A2](../product/user-stories.md#a2--macos-focus--dnd-开启时) | Focus mode 适配（tray 数字变灰） | 用户反馈 Focus mode 下仍干扰 |
| [A3](../product/user-stories.md#a3--笔记本待机后唤醒) | 待机/唤醒主动 refresh | 实测发现唤醒后 polling 卡 ≥ 5s |
| [H3 v0.2+](../product/user-stories.md#h3--不切走也能读到关键信息) | tool_use 完整格式化（按工具类型 syntax highlight） | 用户反馈 `[Bash] git status` 太简略 |

### 1.2 来自 architecture/design (deferred)

| 项 | 内容 | Trigger |
|---|---|---|
| Popup 锚定到 tray icon | 用 NSStatusItem API 把窗口位置锚到 tray | 用户反馈 popup 位置乱跑 |
| Popup 失焦自动 hide | 监听 `WindowEvent::Focused(false)` | 用户反馈点 popup 外不消失烦 |
| Async streaming IPC | invoke 切 emit_event per session（如果 invoke 时长 > 100ms） | benchmark 显示 list_sessions 超 budget |
| fs watcher (notify crate) | 替代 polling | macOS fs event quirk 不再是问题（待 OS upgrade verify） |

### 1.3 来自基础设施

| 项 | 内容 | Trigger |
|---|---|---|
| launchd 守护进程 | app crash 后自动重启 | 用户反馈 crash 后忘记重启 ≥ 3 次 |
| Sign + notarize .app | 申请 Apple Developer Program ($99/年) | 用户反馈 Gatekeeper 摩擦严重 / 收到赞助 / 商业化转向 |
| Homebrew cask 提交 | brew tap | 收到 ≥ 20 GitHub stars |
| CI workflow | benchmark + lint + grep 禁用关键字 | 第一个外部 contributor PR 前 |
| Universal binary (Intel + ARM) | `cargo tauri build --target universal-apple-darwin` | 用户群有 Intel mac |

### 1.4 来自 UX 优化

| 项 | 内容 | Trigger |
|---|---|---|
| 键盘导航 (Tab / Enter / Esc) | popup 内键盘可用 | 用户反馈 trackpad 不爽 |
| Tray icon v1.0 设计 | 找设计师定 icon | release v1.0 前 |
| DMG 安装界面背景 | 自定义 DMG bg image | release v1.0 前 |

### 1.5 永远不做（明确拒绝）

```
（重申，防 v0.2 被诱惑加这些）
- 通知（toast / sound / badge）—— constitution I.1 红线
- 配置面板 —— constitution I.2 红线
- session 命名 / 标签 / 备注 —— constitution I.2
- 跳转到对应终端 tab —— constitution I.4
- 历史已退出 session —— constitution I.3
- 跨平台（Linux / Windows）—— brief 边界
```

要加这些必须先开新 ADR 推翻对应 constitution 条款。
